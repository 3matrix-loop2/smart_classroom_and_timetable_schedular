🖥️ OS Concept Connection

This project mirrors CPU Scheduling in Operating Systems. Just as an OS allocates CPU time to processes without conflicts, this scheduler assigns teachers and classrooms to time slots — preventing "race conditions" (double-booking) using algorithms inspired by Round Robin (fair rotation of subjects across slots) and FCFS (teachers added first get priority assignment).

⚙️ Scheduling Algorithm

  • FCFS: Subjects are queued in the order they're added.
  • Round Robin: Each subject gets one slot per pass, rotating fairly across all days and time slots.
  • Conflict Prevention: Before each assignment, the scheduler checks if the teacher or classroom is already occupied — like a mutex/semaphore in OS.
